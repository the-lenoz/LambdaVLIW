# Язык L_var

Подмножество lisp. Программа является одним S-выражением. 

Единственный тип данных - int32. 
Задаётся положительными константами и бинарными операторами + - * /

Есть оператор *let*:
`(let (var_name (EXPRESSION)) (expression using var_name))` 
результатом этого выражения является результат второго выражения.


результат выполнения программы - результат S-выражения.
