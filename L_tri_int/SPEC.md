# Язык L_tri_int

программа задаётся одним списком (S-expr) примитивных инструкций let:
эти инструкции сопоставляют имя и выражение, состоящее только из одного бинарного оператора и 2 аргументов.
операторы: + - * /
аргументы могут быть либо именем, либо неотрицательной целочисленной константой

тип данных один: int32
