# LambdaVLIW
*Оптимизирующий компилятор типизированного подмножества Lisp/Scheme для e2k, учитывающий особенности архитектуры*

## Проект содержит вспомогательные программы:

- Интерпретаторы промежуточных представлений и их подмножеств
- Интернретаторы подмножеств исходного языка
- Промежуточные компиляторы между парами языков
- Основной компилятор (в разработке)
