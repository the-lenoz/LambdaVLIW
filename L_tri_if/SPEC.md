# Язык L_tri_if


программа задаётся одним списком (S-expr) базовых блоков, 
которые состоят из примитивных инструкций let и условных переходов на базовые блоки.

Инструкции let сопоставляют имя и выражение, состоящее только из одного бинарного оператора и 2 аргументов.
операторы: + - * /
аргументы могут быть либо именем, либо неотрицательной целочисленной константой

Вместо выражения может быть Ф-нода:
`(phi (BB1 id_1) (BB2 id_2))`

тип данных один: int32
