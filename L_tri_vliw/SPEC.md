# Язык L_tri_vliw

программа задаётся одним списком (S-expr) *бандлов* из примитивных инструкций let:
эти инструкции сопоставляют имя и выражение, состоящее только из одного бинарного оператора и 2 аргументов.
операторы: + - * /
аргументы могут быть либо именем, либо неотрицательной целочисленной константой

### бандлы
пачки по <= 6 *независимых* арифметических инструкций:
```lisp
( 
(let a (...))
(let b (...))
...
) ; один бандл 

(
(let x (+ a b))
(let y (...))
...
) ; другой бандл. в нём уже можно использовать результаты первого

```

тип данных один: int32
результат выполнения команды кладётся в *res*
